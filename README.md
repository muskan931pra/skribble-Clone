# skribble-Clone
A real-time multiplayer drawing and guessing game built with React, Node.js and Socket.io

## Live Demo
https://skribble-clone-five.vercel.app/

## Teck Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Real-time: Socket.io

##How to Play
- Enter your name and click Start Game
- One player draws a word, others guess it in the chat
- Correct guess = 10 points
- Player with most points wins
